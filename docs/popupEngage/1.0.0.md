# 1.0.0

| **Release Date**                | May 30, 2016          |
|---------------------------------|------------------------|
| **Plug-In Version**             | 1.0.0                  |
| **Supported nopCommerce Version** | 3.50, 3.60, 3.70     |

**Features:**

1. Pop-up window on customer exit intent when customer has products in the cart.  
2. Subscribe guest customers via pop-up.  
3. Display selected topic page content on pop-up.  
4. Multi-store support.
