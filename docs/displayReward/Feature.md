## Features

- Display reward points on product detail page.
- Automatically calculates reward points for products based on reward point configuration.
- Customizable CSS selector for displaying reward points.
- Customizable content and design for displaying reward points.
- Multi-store support (4.10).
- Multi-language support.

**Supported Version(s):** 3.80, 3.90, 4.00, 4.10
